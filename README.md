## Installing

### Git Clone
```bash
git clone https://github.com/0xl30/Neofetch-Config.git
cd Neofetch-Config
mv ~/.config/neofetch ~/.config/neofetch-bak # Backup old Neofetch config
mv neofetch ~/.config/
nano ~/.zshrc # For bash, use: nano ~/.bashrc
```
